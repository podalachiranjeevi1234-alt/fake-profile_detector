# 🕵️ Fake Profile Detection using Machine Learning

## 🚀 About
This project detects fake social media profiles using ML.

## 🧠 Model
- Random Forest Classifier

## 📊 Features
- Followers
- Following
- Posts
- Bio length
- Profile picture

## ⚙️ Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas

## ▶️ How to Run

### Step 1
pip install -r requirements.txt

### Step 2
python train.py

### Step 3
streamlit run app.py
